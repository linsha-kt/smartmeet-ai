# SmartMeet AI

SmartMeet AI is an AI-powered meeting intelligence platform built with Django. It enables users to upload meeting audio recordings, automatically transcribe them using Whisper, and generate AI-powered meeting summaries and action items using Google Gemini AI.

## Features

- Secure user authentication
- Upload meeting audio files
- Speech-to-text transcription using Whisper
- AI-generated meeting summaries
- Automatic action item extraction
- Meeting history management

## Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- Bootstrap 5
- SQLite
- Whisper
- Google Gemini AI

## Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```
